== Description ==

Counts the title, content, and excerpt characters while you edit your post or page via the admin screen. Even counts the title characters while in Quick Edit!

Decon(structed) Character Counter is a real-time character counter

* for title (post or page)
* for content (post or page)
* for excerpt (post or page)
* for biographical info (user)
* for Quick Edit title (post or page)

Works out of the box and seamlessly integrates with your WordPress admin dashboard so you can focus on writing.

Also comes with a real-time SEO-friendly reminder, just in case.
